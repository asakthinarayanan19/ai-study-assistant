# AI Study Assistant Portal

## Overview

AI Study Assistant Portal is a web-based learning platform that helps students study efficiently using AI-powered tools.

## Features

* AI Chatbot using Groq API
* Notes Management System
* Quiz Generator
* Flashcard Battle
* Dashboard Analytics
* User Login and Registration

## Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* MySQL
* Groq API
* XAMPP

## Installation

1. Install XAMPP.
2. Copy the project folder into `htdocs`.
3. Start Apache and MySQL.
4. Import `study_portal.sql` into phpMyAdmin.
5. Open `http://localhost/study-portal`.

## Project Modules

* Dashboard
* AI Chat Assistant
* Notes
* Quiz Generator
* Flashcard Battle
* User Authentication
